# Running tests

**!!! Run these commands from workspace root dir !!!**

**Run test without coverage:**

```bash
nx e2e frontend-angular-e2e
```

**Run tests with coverage:**

```bash
nx e2e frontend-angular-e2e --configuration=coverage
```
